== Description ==

Accept payment on your GiveWP plugin through multiple payment channels via Lunapay payment gateway.

Signup Lunapay merchant account [here](https://app.lunapay.com/) for live or [here](https://sandbox.lunapay.com/) for sandbox.


== Frequently Asked Questions ==

= Where can i get Client ID, Lunakey and Luna-Signature Key ? =

* Signin Lunapay merchant account

* Make sure account is verified

* Go to the top right of the menu
![first](https://bxcess.sgp1.digitaloceanspaces.com/1.JPG)

* Click the store icon, next click profile button
![second](https://bxcess.sgp1.digitaloceanspaces.com/2.JPG)

* Navigate to API tab
![third](https://bxcess.sgp1.digitaloceanspaces.com/3.JPG)

= What if I have some other question related to Lunapay? =

You may visit our complete FAQ Documentation [here](https://www.lunapay.com/faq-getting-started/) or create a ticket [here](https://lunapay.freshdesk.com/support/login) if there any problem regarding this plugin.
